📚 Project Description
This project creates a BankAccount class where:

Each account has its owner's name and balance.

The total number of accounts and the total balance across all accounts are tracked using static members.

Static member functions are used to display overall bank statistics without needing to access individual objects.

⚙️ Features
Constructor to initialize account details and update statistics.

Destructor to maintain correct statistics when an account is destroyed.

Member functions to deposit, withdraw, and display individual account information.

Static member function to show overall bank statistics (total accounts and total balance).

🛠 Technologies Used
C++

Object-Oriented Programming (OOP) Concepts
